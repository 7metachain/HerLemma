# HerLemma

**谁说女生学不好数学？**

HerLemma 是一个面向高中女生的数学互助平台。AI 帮你把教材翻译成 10 岁小孩都能听懂的话，同伴投票"我听懂了"来验证，所有数据记录在 Avalanche 链上。

不是刷题，不是补课。是女生用自己的方式理解数学，再讲给下一个女生听。

**核心功能：**
- AI 生成三种风格的讲解 + 配图 + 数学动画 + 语音录制
- 知识树串联知识点，点击查看姐妹讲解排行
- "听懂了"投票链上存证，构成不可伪造的教学能力证明
- 榜样学姐档案：她们从这里出发，走进了北大、清华、MIT

**一句话：** 每一条讲解都是一步引理，每一次"听懂了"都是一份证据。她们不需要被谁证明——她们自己就是证明。

**技术栈：** React · Vite · Tailwind · GLM-5 · 豆包 Seedream · Canvas 动画 · Avalanche Fuji
